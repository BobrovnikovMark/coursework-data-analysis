# Глава 3 — Первичный анализ набора данных с изображениями

## Датасет: Horses or Humans Dataset

- **Автор:** Laurence Moroney
- **Источник:** [Kaggle](https://www.kaggle.com/datasets/sanikamal/horses-or-humans-dataset)
- **Лицензия:** CC BY 2.0
- **Тип задачи:** Бинарная классификация (horse / human)
- **Формат:** PNG, 300×300, RGB

## Структура данных

```
horse-or-human/
├── train/
│   ├── horses/    (500 изображений)
│   └── humans/    (527 изображений)
└── validation/
    ├── horses/    (128 изображений)
    └── humans/    (128 изображений)
```

Глава содержит паспорт датасета (описание, разметка, ограничения, обоснование выбора).
Отдельного скрипта анализа нет — глава носит описательный характер.
